Google Analytics Expert

Google Analytics Expert for OpenCart will link your Google Analytics and Google AdWords accounts to your opencart store, so you can view the data below (and much, much more) in the Google Analytics and Google Adwords user interface:

  • Conversion goals (Purchase)
  • Goal Funnel Visualisation (See which step of your pre defined goal the user left the funnel)
  • Product performance (Grouped by product name, you can see quantity purchased, unique purchases, revenue etc)
  • Sales performance (Revenue for sales)
  • Transactions (list of OC Order IDs)
  • Track your store's search queries (View revenue data for each term)
  • Event tracking (track clicks of the add to cart, add to compare & add to wishlist)
  • Multistore support with separate GA tracking code for each website
  • EU cookie compliance to meet EU legislation (Make users aware you use cookies)
  • Option to exclude data you send to Google if logged in as an admin user (To see only customer activity)
  • AdWords conversion value tracking (Let AdWords know when you've made a sale/conversion)
  • AdWords profit reporting (See the actual profit figure from each sale)

All of the above can be viewed/filtered by a date range/period within the Google web interface

Setting up the goal funnel

Click on Goals in the left pane and enter the settings as shown in the image below (scroll below the image to copy & paste into analytics):



For goal destination, select Regular Expression and enter this:
/index\.php\?route=*checkout/success*

You will need to adjust the goal funnel steps according to what theme or custom checkout you are using. The list below covers the most common set ups, adjust the goal funnel steps accordingly:

Default/Standard Checkout
  Name Screen/Page
Step 1 Account /gae/checkout-options
Step 2 Billing Address /gae/payment-address
Step 3 Shipping Address /gae/shipping-address
Step 4 Shipping Method /gae/shipping-method
Step 5 Payment Method /gae/payment-method
Step 6 Confirm Order /gae/confirm

Do you have a 3rd party one page checkout extension and you want to use a goal funnel to track checkout activity? Please send a support request in the extension settings for a quote.

Google Adwords - Creating a new conversion tag

  1. Login to AdWords
     
  2. Click Tools > Conversions

     
  3. Click the red button to add a new conversion

     
  4. Select the 'Website' from the options

     
  5. Setup your conversion by copying the settings from the image below

     
  6. Click the 'Save and continue' button
     
  7. Scroll down to the 'Tag for Purchase' code and copy the value of the google_conversion_id and google_conversion_label variables (excluding quotes) to the relevant Adwords Settings tab in the extension settings

Google Adwords - Use an existing conversion tag

  1. Login to AdWords
     
  2. Click Tools > Conversions

     
  3. Click on the name of the conversion (must be set up as a purchase conversion or you will need to create a new one)
     
  4. Scroll down to the code for your existing conversion and copy the value of the google_conversion_id and google_conversion_label variables (excluding quotes) to a notepad for use in the installation guide later

EU Cookie Consent

Set up is simple. Just choose to Enable or Disable the feature in the Google Analytics Expert settings.

The EU Cookie Consent feature is using an open source solution from Insites which I have included in the Google Analytics Expert extension.

This is a very minimalistic solution to ensure compliance. The message will show at the bottom of the browser window and it looks like this:

The message will display until the user clicks the 'Got it!' button. At which point the message won't show again until the user clears your site cookies from their browser or 10 years has passed, whichever comes first!
